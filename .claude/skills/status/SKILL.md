# /status — Project Status Check

Check the current study project for completeness and offer to help fill gaps.

## Steps

1. Read STUDY-PROTOCOL.md — is it filled in?
2. Read METHODS-REGISTER.md — are there methods registered?
3. Read DECISION-LOG.md — are decisions being recorded?
4. Check methods/ directory — what work exists?
5. For each gap found, suggest the relevant craft skill.

## Output

Produce a brief status report:
- What exists and looks good
- What's missing or incomplete
- For each gap: what skill can help (e.g., "Run /new-study to fill in your study protocol")
