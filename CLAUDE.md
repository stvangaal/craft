# CLAUDE.md — craft

## Project Overview

CRAFT (Community for Reproducible AI Frameworks and Tools) is a toolkit for reproducible AI-assisted research. This repo provides workflows and skills for bootstrapping and maintaining research study projects.

## Key Documents

| Document | Purpose |
|---|---|
| `PHILOSOPHY.md` | The four beliefs (from CRAFT charter) |
| `CRAFT-WORKFLOW.md` | Governing rules for craft projects |
| `learn/` | Self-directed learning pathway for clinician-researchers |
| `.claude/skills/` | Machine skills (Claude Code slash commands) |

## CLI Usage

```bash
# Bootstrap a new study project
bin/craft bootstrap ~/Projects/my-study

# Update an existing study project (run from within the project)
./craft update
```
