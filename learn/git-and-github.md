# Git & GitHub

Git is a version control system that tracks every change you make to your files, who made it, and when. GitHub is a platform for hosting Git repositories online, making it possible to collaborate, share work, and maintain a complete history of a project. If you've ever emailed yourself a file called `final_v3_REAL_final.docx`, version control solves that problem permanently.

In research, Git provides an audit trail. Every analysis script, data cleaning step, and configuration change is recorded with a timestamp and a description. This makes your work reproducible — you can always go back to the exact state of your code at any point. GitHub adds collaboration: co-investigators can work on the same project without overwriting each other's changes, and reviewers can see exactly what changed and why.

CRAFT projects are Git repositories hosted on GitHub. Understanding the basics — commits, branches, push, pull — is essential for participating in any CRAFT workflow.

## Getting Started

Git is a command-line tool. GitHub is a web platform with a free tier that covers everything you need for research.

| Resource | Notes |
|---|---|
| [Git Downloads](https://git-scm.com/downloads) | Official Git installer for macOS, Windows, and Linux |
| [GitHub](https://github.com) | Create a free account. GitHub Free includes unlimited public and private repositories |
| [GitHub Desktop](https://desktop.github.com) | Optional GUI for Git. Useful if you prefer visual interfaces while learning, but CRAFT workflows use the command line |
