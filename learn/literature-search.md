# Literature Search

A literature search is the systematic process of finding all relevant published research on a topic. In evidence-based research, the search itself is a method — it needs to be documented, reproducible, and transparent enough that someone else could run the same search and get the same results. This is what separates a rigorous systematic review from a casual Google Scholar session.

AI tools can accelerate literature search from weeks to hours, but speed without structure produces noise. The value of AI in this workflow is not just finding papers faster — it's screening thousands of abstracts, identifying patterns across studies, and extracting structured data from unstructured text. The risk is that AI makes it easy to skip the methodological discipline that makes a search trustworthy.

This topic covers how to formulate reproducible search strategies, use AI-powered tools to execute and screen them, and document what you did so others can verify it. If your research involves a systematic review, scoping review, or any structured evidence synthesis, these are foundational skills.

## Getting Started

Literature search doesn't require software installation, but it does require access to bibliographic databases and an understanding of the PRISMA framework for reporting.

| Resource | Notes |
|---|---|
| [PubMed](https://pubmed.ncbi.nlm.nih.gov) | Free. The primary database for biomedical literature. Supports structured search with MeSH terms, Boolean operators, and filters |
| [PRISMA 2020 Statement](https://www.prisma-statement.org) | The reporting guideline for systematic reviews. Read the checklist and flow diagram before designing your first search |

## Quick References

- [PRISMA-S Checklist](https://doi.org/10.1186/s13643-020-01542-z) — 16-item checklist specifically for reporting literature searches reproducibly; covers databases searched, search strategies, limits applied, and date ranges
- [PubMed Search Guide](https://pubmed.ncbi.nlm.nih.gov/help/) — official guide to PubMed search syntax, MeSH terms, Boolean operators, and advanced search features
- [Cochrane Handbook: Searching for Studies](https://training.cochrane.org/handbook/current/chapter-04) — the authoritative guide to designing search strategies for systematic reviews

## Tools

AI-powered tools that assist with different stages of the literature search workflow:

| Tool | Purpose | Notes |
|---|---|---|
| [Elicit](https://elicit.com) | Search and extraction | Searches academic databases, summarizes papers, and extracts data into structured tables |
| [Rayyan](https://www.rayyan.ai) | Screening | Collaborative screening platform with AI-assisted relevance detection. Reduces screening time significantly |
| [Semantic Scholar](https://www.semanticscholar.org) | Discovery | Free AI-powered academic search engine with TLDR summaries and citation analysis |
| [ResearchRabbit](https://www.researchrabbit.ai) | Discovery | Visualizes connections between papers and discovers related work by expanding from seed articles |
