# Markdown

Markdown is a lightweight plain-text formatting language. It lets you write structured documents — headings, lists, links, tables, code blocks — using simple syntax that is readable even before it's rendered. In research, Markdown is the standard for documentation in code-based projects: README files, protocols, decision logs, and workflow descriptions are all written in it. Every file in CRAFT that isn't code is Markdown.
