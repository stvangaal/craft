# Markdown

Markdown is a lightweight plain-text formatting language. It lets you write structured documents — headings, lists, links, tables, code blocks — using simple syntax that is readable even before it's rendered. In research, Markdown is the standard for documentation in code-based projects: README files, protocols, decision logs, and workflow descriptions are all written in it. Every file in CRAFT that isn't code is Markdown.

You can learn the basics in under an hour. The syntax is designed to be intuitive — a line starting with `#` is a heading, `**bold**` is bold, `-` starts a list item. If you've written a bulleted list in an email, you already understand the idea. The payoff is that your documents become version-controllable, platform-independent, and renderable in tools like GitHub, VS Code, and Obsidian without any special software.

## Getting Started

Markdown doesn't require installation — you can write it in any text editor. That said, a good editor makes the experience much better with live preview, syntax highlighting, and shortcuts.

| Application | Platform | Notes |
|---|---|---|
| [VS Code](https://code.visualstudio.com) | macOS, Windows, Linux | Free. Built-in Markdown preview (Cmd/Ctrl+Shift+V). The recommended editor for CRAFT projects since you'll already use it for code. |
| [Obsidian](https://obsidian.md) | macOS, Windows, Linux | Free for personal use. Excellent for research notes and knowledge management. Live preview and wiki-style linking between documents. |
| [MacDown](https://macdown.uranusjr.com) | macOS | Free. Lightweight split-pane editor — write on the left, see rendered output on the right. |
| [Typora](https://typora.io) | macOS, Windows, Linux | Paid ($15). Clean WYSIWYG-style editor where formatting renders inline as you type. |
