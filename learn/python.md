# Python

Python is a general-purpose programming language that has become the default for data work in research. It has mature libraries for statistics, data manipulation, machine learning, natural language processing, and visualization. If your research involves analyzing data, building pipelines, or working with AI tools, Python is almost certainly the language those tools are written in.

You don't need to become a software engineer. What you need is enough fluency to read Python, understand its logic, and modify AI-generated code. When Claude Code writes a data extraction script or an analysis pipeline, you need to be able to review what it did, catch mistakes, and adjust parameters. That requires understanding variables, data types, control flow, functions, and how to work with libraries like pandas and NumPy.

Python is also the language of most AI/ML frameworks (PyTorch, scikit-learn, Hugging Face), so understanding it gives you direct access to the tools driving AI-assisted research.

## Getting Started

Install Python from the official source. Most macOS and Linux systems have a system Python, but you should install a separate version for development to avoid conflicts.

| Resource | Notes |
|---|---|
| [python.org/downloads](https://www.python.org/downloads/) | Official Python installer for all platforms. Download the latest stable release (Python 3.x) |
| [Anaconda](https://www.anaconda.com/download) | Alternative installer that bundles Python with data science libraries (pandas, NumPy, Jupyter). Heavier but convenient if you want everything in one install |
