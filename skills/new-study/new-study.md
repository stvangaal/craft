# /new-study — Create a Study Protocol

Guide the researcher through filling in STUDY-PROTOCOL.md by asking questions one at a time.

## Steps

1. Read the current STUDY-PROTOCOL.md
2. For each empty section, ask the researcher a focused question
3. Fill in their answers, asking follow-up questions as needed
4. When complete, summarize what was captured
5. Offer to record the study setup as a decision in DECISION-LOG.md
