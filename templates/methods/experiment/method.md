# Method: Experiment / Observational Study

## Purpose

Design, execute, and analyze a research study with pre-specified methods.

## Steps

1. **Hypothesis** — State the research question and expected findings
2. **Protocol** — Write the study protocol (use `STUDY-PROTOCOL.md`)
3. **Analysis plan** — Pre-specify statistical methods (`/analysis-plan`)
4. **Data collection** — Gather data per protocol
5. **Analysis** — Execute the analysis plan (`/run-analysis`)
6. **Interpretation** — Contextualize findings

## Key Artifacts

- Study protocol
- Statistical analysis plan
- Results notebook / output
- Interpretation notes

## What "Reproducible" Means

The analysis plan is pre-registered before data analysis begins. Someone else can re-run the analysis code and get the same results.

## Skills

- `/analysis-plan` — build a statistical analysis plan
- `/run-analysis` — execute and log an analysis step
