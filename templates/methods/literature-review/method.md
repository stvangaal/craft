# Method: Literature Review

## Purpose

Systematically search, screen, and synthesize published evidence to answer a specific research question.

## Steps

1. **Question formulation** — Define the research question (PICO or equivalent)
2. **Search strategy** — Build database queries (`/search-strategy`)
3. **Screening** — Apply inclusion/exclusion criteria (`/screen`)
4. **Data extraction** — Extract structured findings (`/evidence-table`)
5. **Synthesis** — Summarize and interpret findings

## Key Artifacts

- Search protocol (databases, queries, date of search)
- Inclusion/exclusion criteria
- PRISMA flow diagram
- Evidence table

## What "Reproducible" Means

Another reviewer can re-run the search strategy and get substantially the same corpus. Screening decisions are documented. Extraction is structured and auditable.

## Skills

- `/search-strategy` — build database search queries
- `/screen` — set up screening workflow
- `/evidence-table` — extract structured findings
